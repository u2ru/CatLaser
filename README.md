# CatLaser
Wireless network auditor/pentesting tool
Created for Kali Linux

## Usage (one line command)
```
git clone https://github.com/u2ru/CatLaser.git && cd CatLaser/ && sudo chmod +x ./catlaser.sh && sudo ./catlaser.sh
```
